# 🚀 КАК ЭКСПОРТИРОВАТЬ ПРОЕКТ ИЗ FIGMA MAKE

## ✅ ВЫ ВИДИТЕ КНОПКУ "SHARE" - ЭТО ОНА!

**ДА, НАЖИМАЙТЕ НА "SHARE"!** 

В Figma Make кнопка экспорта часто называется **"Share"** (Поделиться).

---

## 📱 ПОШАГОВАЯ ИНСТРУКЦИЯ:

### **ШАГ 1: НАЙДИТЕ КНОПКУ "SHARE"**

**Где находится:**
- В правом верхнем углу экрана
- Рядом с названием проекта
- Обычно синяя или фиолетовая кнопка

**Как выглядит:**
```
┌─────────────────────────────┐
│  Figma Make    [Share] ⬇️   │ ← Здесь!
└─────────────────────────────┘
```

---

### **ШАГ 2: НАЖМИТЕ НА "SHARE"**

После нажатия откроется меню с опциями:

```
┌──────────────────────────┐
│  Share                   │
│                          │
│  📤 Export Code          │ ← Это!
│  🔗 Share Link           │
│  👥 Invite Team          │
│  ⚙️  Settings            │
└──────────────────────────┘
```

**Выберите:** **"Export Code"** или **"Download"** или **"Export Project"**

---

### **ШАГ 3: ВЫБЕРИТЕ ФОРМАТ**

Могут быть варианты:

```
┌──────────────────────────┐
│  Export Project          │
│                          │
│  ✓ Download as ZIP       │ ← Выберите это!
│  ○ Copy to GitHub        │
│  ○ Deploy to Vercel      │
└──────────────────────────┘
```

**Выберите:** **"Download as ZIP"** или **"Download Project"**

---

### **ШАГ 4: НАЖМИТЕ "DOWNLOAD" ИЛИ "EXPORT"**

```
┌──────────────────────────┐
│  Export as ZIP           │
│                          │
│  Project: bento-torty    │
│  Size: ~2 MB             │
│                          │
│  [Download] [Cancel]     │
│     👆 ЖМУ СЮДА!         │
└──────────────────────────┘
```

**Файл начнёт скачиваться!** 📦

---

## 🎯 АЛЬТЕРНАТИВНЫЕ ВАРИАНТЫ:

### **ВАРИАНТ А: Кнопка с иконкой стрелки**

Если не видите "Share", ищите:

```
[⬇️] ← Иконка стрелки вниз
```

Обычно в правом верхнем углу.

---

### **ВАРИАНТ Б: Меню с тремя точками**

Ищите иконку:

```
[⋮] ← Три точки вертикально
или
[···] ← Три точки горизонтально
```

Нажмите → выберите **"Export"** или **"Download"**

---

### **ВАРИАНТ В: Прямой деплой на Vercel**

**ЕСЛИ в меню "Share" есть опция "Deploy to Vercel" - ЭТО ЕЩЁ ПРОЩЕ!**

```
┌──────────────────────────┐
│  Share                   │
│                          │
│  🚀 Deploy to Vercel     │ ← Можете нажать сразу!
└──────────────────────────┘
```

**Что произойдёт:**
1. Figma Make автоматически подключится к Vercel
2. Попросит авторизоваться в Vercel
3. Автоматически задеплоит проект
4. Выдаст готовую ссылку!

**Это самый простой способ! Попробуйте его!**

- **Publish** (Опубликовать)

### **Иконки:**
- ⬇️ Стрелка вниз
- 📤 Стрелка вверх из коробки
- 🚀 Ракета
- ⋮ Три точки
- ··· Три точки

## 🎁 ЕСЛИ ЕСТЬ "DEPLOY TO VERCEL" - ВАМ ПОВЕЗЛО!

### **Это САМЫЙ простой путь:**

1. **Нажмите "Share"**
2. **Выберите "Deploy to Vercel"**
3. **Авторизуйтесь через GitHub или Email**
4. **Подождите 1 минуту**
5. **Получите ссылку!**

**Готово! Не нужно скачивать и загружать вручную!** 🎉

## 💡 ЕСЛИ ОТКРОЕТСЯ "DEPLOY TO VERCEL":

### **Это значит, что Figma Make интегрирован с Vercel!**

**Что делать:**
1. Нажмите "Deploy to Vercel"
2. Разрешите доступ (если попросит)
3. Выберите "Continue with GitHub" или "Continue with Email"
4. Подождите 30-60 секунд
5. **ПОЛУЧИТЕ ССЫЛКУ!** 🎉

**Это НАМНОГО проще, чем скачивать и загружать вручную!**
